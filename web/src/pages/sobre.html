Sobre

Meu Time é um projeto colaborativo com o objetivo de informar a infraestrutura e grande parte das trâmites do maior e mais popular esporte do mundo.

Para frontend o projeto se utiliza de html, css de funções javascript
Para backend o projeto se utiliza do banco de dados postgres, sequelize para criação dos models e express para o roteamento da api.
